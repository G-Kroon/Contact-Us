# Contact Us - Static Site

## Overview
A simple static contact form page with EmailJS integration for sending messages.

## Project Structure
- `index.html` - Main contact form page
- `custom.css` - Custom styles
- `form-validation.js` - Client-side form validation logic
- `emailjs-integration.js` - EmailJS email sending integration
- `gk.png`, `gk1.png`, `gk2.jpeg` - Image assets
- `robots.txt` - SEO robots file

## Tech Stack
- Plain HTML, CSS, JavaScript (no build system)
- Bootstrap (dark theme via CDN)
- Font Awesome icons (via CDN)
- EmailJS for form submission

## Running the Project
Served as a static site using `npx serve` on port 5000.

## Deployment
Configured as a static deployment (publicDir: ".").
